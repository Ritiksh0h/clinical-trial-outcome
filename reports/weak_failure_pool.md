## Recommendation

Across phases the weak set adds **15,599 new real failure examples** vs **7,668** already in gold (min per-phase ratio 1.7×).

**There is substantial extra failure signal — weak+gold with covariate correction is worth trying as a challenger.** The non-completed weak trials add 1.7–2.3× the gold failure pool per phase and are ~99% trustworthy. But note both naive mixes distort the class prior in opposite directions: dumping in all weak data swings it success-heavy (the population shift returns, ~90k extra successes); adding only weak failures swings it failure-heavy (e.g. Phase I goes from 19% to ~8% positive). The principled route is weak+gold with importance weighting toward the gold/deployment covariate distribution, class prior fixed to gold base rates (scale_pos_weight as already specified). Keep gold-only as the baseline and only promote the weak-augmented challenger if it beats it on the gold-eval PR-AUC gate.
