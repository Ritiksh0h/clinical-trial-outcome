## C — Conclusion

### Gold label definition

COMPLETED trials success rate: **91.2%**

Definition: **COMPLETION / PROGRESSION DEFINITION** — most completed trials are labeled success
Explanation: Completing the trial protocol ≈ success. This is an easier, less clinically meaningful target.

### CTO weak label definition

Phase 1 weak COMPLETED success rate: 99.9%
Phase 2 weak COMPLETED success rate: 99.9%
Phase 3 weak COMPLETED success rate: 100.0%

Definition: **COMPLETION DEFINITION** — CTO weak labels treat COMPLETED ≈ success

### Gap diagnosis

| Source | Definition | Positive rate driven by |
|--------|-----------|--------------------------|
| Gold labels | Completion-vs-termination | Completion rate of the gold population |
| CTO weak labels | Completion-vs-termination | Completion rate of the weak population |

Both label sources encode the **same** target: COMPLETED ≈ success, TERMINATED/WITHDRAWN
≈ failure. The positive-rate gap between them is therefore NOT a definitional mismatch —
it is a difference in the two populations' completion rates.

### Cross-reference: consistent with the label-vs-population diagnostic

This finding is consistent with `reports/label_vs_population.md` and
`reports/weak_failure_pool.md`. Because both label sources encode completion-vs-termination,
they **agree 96–98% on the same (overlap) trials** — the pure label effect is only ~3pp,
while the population effect is ~45pp (94% of the gap). The tiny residual disagreement is the
~9–11% of weak "successes" the experts downgraded (completed-but-weak-on-efficacy); the
weak=0 / gold=1 cell is exactly 0 in all phases, so the weak LF never calls a real success a
failure. The gap is population-driven (covariate shift), not label-driven.

### Implications for Phase 2

1. **Both label sources are trustworthy; the choice is a population/prior question, not a
   "which labels are correct" question.** Gold-only training is the clean baseline. Weak+gold
   with covariate correction is a viable challenger (the weak set adds ~15.6k trustworthy
   failure examples, 1.7–2.3× the gold failure pool) but must beat gold-only on the gold-eval
   PR-AUC gate.

2. **The gap is population/covariate shift, not supervision leakage.** The Phase 1 model was
   trained on a ~83%-completed population and evaluated on a ~20%-completed one. It did not
   learn labeling-function artifacts.

3. **The target is completion-vs-termination, not efficacy — frame results honestly.**
   Gold Phase I pos_rate=0.199 is correct for the gold set's population, but that population
   is enriched for TERMINATED/WITHDRAWN trials; it is not a representative base rate, and it
   should NOT be equated with the FDA ~14% Phase I→approval figure (different, harder target).

4. **Do not compute scale_pos_weight from weak-set positive rates.**
   Use gold base rates as specified in CLAUDE.md:
   Phase I ≈ 4.02, Phase II ≈ 2.23, Phase III ≈ 0.89.
